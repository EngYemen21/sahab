# موقع تدفق

نسخة صفحة هبوط عربية متجاوبة بالكامل، تعمل مباشرة دون حاجة إلى إعدادات.

## التشغيل
افتح ملف `index.html` في المتصفح.

## التعديل السريع
- بيانات العملاء موجودة داخل قسم `#clients`.
- البريد المستلم لنموذج التواصل موجود في نهاية الملف داخل `mailto:`.
- الألوان الرئيسية موجودة أعلى CSS داخل `:root`.
- يمكن استبدال شعار حرف «ت» بالشعار الرسمي لاحقًا.

## ملاحظة
نموذج التواصل يفتح برنامج البريد برسالة جاهزة؛ ربطه بقاعدة بيانات أو CRM يحتاج Backend أو خدمة نماذج خارجية.
